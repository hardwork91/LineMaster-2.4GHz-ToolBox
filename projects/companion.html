Companion Tool

Browser-based utility to emulate your LineMaster handle on screen, edit model profiles and sync SD card configuration for Typhoon and Zephyr handles.

Browser compatibility: Requires Chrome or Edge on desktop (Web Serial API). Not available on mobile browsers or Safari.

About this Project

The Companion Tool connects to your LineMaster handle via USB using the Web Serial API. It provides a visual interface to view and edit model profiles, configure settings and manage SD card contents — all without installing any software.

This tool is currently in beta. Features are being actively developed and refined based on pilot feedback.

Planned Features

Handle Emulation

See your handle's display and controls rendered on screen in real time.

Model Profile Editor

Create, edit and organize model profiles with a visual interface.

SD Card Sync

Read and write configuration files on your handle's SD card directly from the browser.
